📱 DoDo 앱이란?

DoDo는 다음을 도와주는 스마트한 할 일 관리 앱입니다:

✅ 할 일 관리

할 일을 쉽게 추가하고 완료 체크

📅 일정 관리

마감일 설정 및 자동 알림

🔔 알림 기능

잊지 말아야 할 일을 시간에 맞춰

📸 파일 첨부

사진이나 문서를 할 일과 함께 저장

🌐 클라우드 동기화

여러 기기에서 같은 할 일 목록 사용

📊 통계 기능

내 생산성을 한눈에 볼 수 있음

🎯 DoDo의 특징

  • 완전 무료 - 광고 없음(추후 광고 기제)
  • 간단한 인터페이스 - 누구나 쉽게 사용
  • 강력한 기능 - 복잡한 일도 관리 가능
  • 안전한 보안 - 당신의 데이터는 안전
DoDo는 "할 일을 하고 싶은 욕구(Do)"를 뜻하는 이름입니다. 더 많은 일을 완료하고 싶은 당신을 위해 만들어졌습니다!

🎯 주요 기능

기능 설명 활용 예
📝 할 일 CRUD 할 일 추가, 수정, 삭제 "운동하기", "보고서 작성"
🏷️ 카테고리 일, 생활, 취미 등으로 분류 일과 생활을 구분해서 관리
🔁 반복 설정 매일, 매주, 매월 반복 "매일 운동", "매주 청소"
📌 서브태스크 큰 일을 작은 단계로 분해 프로젝트를 세부 작업으로 분할
⏰ 알림 설정 예정 시간에 자동 알림 중요한 일을 빠뜨리지 않음
🤐 스누즈 알림을 나중에 받기 "5분 후 다시 알림"
📍 위치 알림 특정 장소에서 알림 "카페에 도착하면 알림"
📁 파일 첨부 이미지, PDF, 문서 저장 "회의록.pdf" 함께 저장
🔄 오프라인 모드 인터넷 없이도 사용 비행기 타면서도 작업 가능
📊 통계 생산성 분석 "이번 주 완료율: 85%"
📱 홈 화면 위젯 홈 화면에서 할 일 확인 앱 실행 없이 할 일 보기

🚀 시작하기

1️⃣ 앱 다운로드 및 설치

Android 사용자

Step 1: Google Play Store 열기

당신의 Android 휴대폰에서 Google Play Store 앱을 엽니다.

Step 2: "DoDo" 검색하기

검색창에 "DoDo 할 일 관리" 입력

Step 3: 설치 버튼 클릭

파란색 "설치" 버튼을 탭합니다.

Step 4: 설치 완료

설치가 완료될 때까지 기다린 후 "열기" 버튼을 탭합니다.

iOS 사용자(개발중)

Step 1: App Store 열기

당신의 iPhone에서 App Store를 엽니다.

Step 2: "DoDo" 검색하기

검색 탭에서 "DoDo 할 일 관리" 입력

Step 3: 다운로드

구름 아이콘을 탭하여 다운로드합니다.

Step 4: Face ID/Touch ID 인증

생체 인증으로 설치를 확인합니다.

2️⃣ 계정 만들기

⭐⭐⭐ 추천: Google로 가입 (가장 쉬움)

Step 1: "Google로 로그인" 클릭

화면에 보이는 파란색 버튼을 탭합니다.

Step 2: 자신의 Google 계정 선택

이미 로그인한 계정이 있다면 바로 선택할 수 있습니다.

Step 3: "허용" 클릭

DoDo에 접근할 권한을 허용합니다.

자동으로 DoDo 계정이 생성됩니다! 어려울 것이 없습니다!

이메일로 가입

Step 1: "이메일로 가입하기" 클릭

Google 버튼 아래의 다른 옵션을 선택합니다.

Step 2: 이메일 입력

자주 사용하는 이메일 (예: myemail@gmail.com)

Step 3: 비밀번호 설정

8글자 이상, 숫자 포함 권장

Step 4: "가입" 클릭

계정이 생성되고 자동으로 로그인됩니다.

자주 사용하는 이메일을 사용하세요. 나중에 다른 기기에서도 같은 이메일로 로그인할 수 있습니다!

3️⃣ 첫 할 일 추가하기

로그인하면 할 일 목록 화면이 보입니다. 이제 첫 번째 할 일을 추가해봅시다!

화면 하단의 파란 "+" 버튼 또는 상단의 "새 할 일 추가"를 클릭하세요!

단계별 진행

Step 1: 제목 입력 (필수)

예: "운동하기", "보고서 작성", "장보기"

Step 2: 설명 입력 (선택사항)

예: "30분 조깅", "월요일까지 제출"

Step 3: 추가 설정
  • 🏷️ 카테고리 - "건강" 선택
  • 📅 마감일 - "2025-12-10" 선택
  • ⏰ 알림 시간 - "오후 6:00" 선택
  • 🔁 반복 - "매일" 또는 "매주 목요일"
Step 4: "저장" 버튼 클릭

할 일이 목록에 추가됩니다!

축하합니다! 첫 번째 할 일을 생성했습니다! 이제 DoDo를 사용할 준비가 완료되었습니다!

💡 일상 사용법

📋 할 일 완료하기

1. 할 일 목록에서 항목을 찾기 2. 왼쪽의 체크박스 ☐ 클릭 3. ✅로 변하면서 완료됨!
완료한 할 일은 자동으로 통계에 반영됩니다!

✏️ 할 일 수정하기

1. 수정하려는 할 일을 길게 누르거나 탭 2. "수정" 또는 "편집" 메뉴 선택 3. 원하는 내용 변경 4. "저장" 클릭

🗑️ 할 일 삭제하기

1. 삭제할 할 일을 길게 누르기 2. "삭제" 또는 "휴지통" 아이콘 클릭 3. 확인 창에서 "삭제" 다시 클릭 4. 할 일이 삭제됨
삭제된 할 일은 복구할 수 없습니다!

🏷️ 카테고리 관리하기

카테고리는 할 일을 분류하는 방법입니다.

카테고리 만들기

1. 설정 또는 메뉴에서 "카테고리" 클릭 2. "새 카테고리" 또는 "+" 버튼 클릭 3. 카테고리 이름 입력 (예: "일", "생활", "취미") 4. 색상 선택 (선택사항) 5. "생성" 또는 "추가" 클릭

추천 카테고리

📊 일

업무 관련

🏠 생활

청소, 요리, 장보기

💪 건강

운동, 식단

🎮 취미

게임, 독서, 영화

🎓 공부

학습, 숙제

🔔 알림 설정하기

알림은 정해진 시간에 자동으로 알려줍니다.

알림 시간 설정

1. 할 일 추가/수정 화면에서 "알림 시간" 탭 2. 시간 선택 (예: "오후 3:00") 3. "확인" 또는 "저장" 클릭 4. 그 시간이 되면 핸드폰이 울립니다!

알림 스누즈 (잠깐 미루기)

화면의 "스누즈" 또는 "5분 후" 버튼 클릭 → 5분, 10분, 30분, 1시간 등 선택 가능

📍 위치 기반 알림

"특정 장소에 가면 알림이 울린다"는 뜻입니다.

위치 알림 설정

1. 할 일 추가/수정 화면에서 "위치" 탭 2. 지도에서 위치 검색 - "카페 근처" → 검색 - "회사" → 검색 3. 원하는 위치 선택 4. 반경 설정 (기본값: 500m) 5. "저장" 클릭

활용 예

  • 회사에 도착하면 "오늘 할 일 확인" 알림
  • 마트 근처에 가면 "장보기" 알림
  • 카페에 도착하면 "커피 주문" 알림

🔄 반복 설정하기

같은 일을 자동으로 반복할 수 있습니다.

1. 할 일 추가 화면에서 "반복" 탭 2. 반복 주기 선택: - 매일 (아침마다 운동) - 매주 월요일 (주간 회의) - 매월 10일 (월급일) - 커스텀 (자신만의 패턴) 3. "저장" 클릭

📎 파일 첨부하기

할 일과 함께 사진이나 파일을 저장할 수 있습니다.

1. 할 일 추가/수정 화면에서 "파일" 또는 "📎" 아이콘 2. 선택 옵션: - 📷 카메라 - 사진 찍기 - 🖼️ 갤러리 - 저장된 사진 선택 - 📄 파일 - 문서/PDF 선택 3. 파일 선택 후 "확인" 클릭 4. "저장" 클릭

활용 예

  • 영수증 사진 - "물품 구매" 할 일에 첨부
  • 회의록 PDF - "회의 참석" 할 일에 첨부
  • 찍은 사진 - "사진 편집" 할 일에 첨부

📊 통계 보기

당신의 생산성을 한눈에 볼 수 있습니다.

1. 앱 메인 화면의 "통계" 또는 "📊" 탭 클릭 2. 이번 주 완료 현황 확인 3. 요일별 완료율 확인 4. 카테고리별 진행 상황 확인

보이는 정보

  • 📈 이번 주 완료율 - 몇 개 완료했는지
  • 📊 카테고리별 진행 - 어느 분야에서 잘하고 있는지
  • 🔄 요일별 패턴 - 언제 가장 생산성이 높은지
  • ⭐ 현재 스트릭 - 며칠 연속으로 완료했는지

📱 홈 화면 위젯 사용하기

앱을 열지 않고도 홈 화면에서 바로 할 일을 확인하고 완료할 수 있습니다!

위젯 추가하기 (Android)

1. 홈 화면 빈 곳을 길게 누르기 2. "위젯" 또는 "Widgets" 선택 3. "DoDo" 검색 또는 찾기 4. 원하는 위젯 선택: - 📋 할일 위젯 - 오늘 할 일 목록 (최대 3개) - 📅 캘린더 위젯 - 월간 캘린더 5. 홈 화면에 위젯 배치

📋 오늘의 할일 위젯 기능

✅ 체크박스로 완료 처리

위젯에서 직접 체크박스를 눌러 할 일 완료! 앱을 열 필요 없이 바로 완료 처리됩니다.

➕ 빠른 할일 추가

위젯의 + 버튼을 눌러 새 할일 추가 화면으로 바로 이동

📅 오늘 할일만 표시

마감일이 오늘이거나, 오늘 생성한 할일만 표시됩니다.

🔄 실시간 동기화

위젯에서 완료한 내용이 클라우드에 자동 동기화됩니다.

위젯에서 체크박스를 누르면 "✓ 완료!" 메시지와 함께 바로 처리됩니다. 앱이 실행 중이면 클라우드에도 자동 동기화!

위젯 테마 변경하기

1. DoDo 앱 실행 2. 설정 → "위젯 설정" 선택 3. 원하는 위젯 유형 선택 4. 테마 선택: - 🤍 라이트 - 밝은 배경 - 🖤 다크 - 어두운 배경 - 🔲 투명 - 배경이 투명 - 💙 블루 - 파란색 배경 - 💜 퍼플 - 보라색 배경 5. 테마 선택 후 자동 적용!
투명 테마는 배경화면이 보여서 예쁘게 꾸밀 수 있어요!

📡 오프라인 모드

인터넷이 없어도 할 일을 관리할 수 있습니다!

오프라인 모드 작동 방식

인터넷 연결됨: ✅ 자동으로 클라우드에 저장

인터넷 끊김: 📱 휴대폰에 임시 저장 → 연결되면 자동 동기화

오프라인 상태 표시

- 화면 상단에 주황색 배너 표시 - "오프라인 모드 - 변경사항은 로컬에 저장됩니다" - 연결되면 자동으로 동기화 시작

동기화 상태 확인

  • ✅ 초록색 체크 - 동기화 완료
  • 🔄 회전 아이콘 - 동기화 진행 중
  • ⚠️ 주황색 아이콘 - 오프라인 상태
  • ❌ 빨간색 아이콘 - 동기화 실패 (재시도 버튼 클릭)
오프라인에서 작업한 내용은 인터넷이 연결되면 자동으로 동기화됩니다!

🌐 여러 기기에서 동기화

스마트폰, 태블릿, 컴퓨터에서 같은 할 일 목록을 사용할 수 있습니다!

동기화 원리:

핸드폰 ←→ 클라우드 ←→ 컴퓨터

(자동 동기화)

다른 기기에서 로그인

1. 다른 기기에서 DoDo 앱 설치 2. "로그인" 선택 3. 같은 이메일/Google 계정으로 로그인 4. 자동으로 모든 할 일이 나타남!

🎓 고급 기능

원하는 사람들을 위한 고급 기능들입니다. 이 기능들은 선택사항이므로 필요할 때만 사용하면 됩니다.

🎯 프로젝트 관리

큰 프로젝트를 작은 단계(서브태스크)로 나누기

예: "웹사이트 만들기"
  • ✅ 디자인 만들기
  • ⏳ 데이터베이스 설정하기
  • ⏳ 코드 작성하기
  • ⏳ 테스트 및 배포하기

📈 생산성 분석

통계 기능을 활용해 자신의 생산성 패턴을 이해하고 개선할 수 있습니다.

예:

  • 팀 프로젝트 공유
  • 가족 할 일 목록
  • 실시간 협업

🆘 자주 묻는 질문

Q: 로그인할 수 없어요

다음을 확인하세요:

  • 이메일 주소/비밀번호가 맞는지 확인
  • 인터넷 연결 확인
  • 앱을 완전히 종료했다가 다시 실행
  • "비밀번호 찾기" 사용
Q: 모든 할 일이 사라졌어요

다음을 시도하세요:

  • 앱 다시 시작
  • 로그아웃 후 다시 로그인
  • 인터넷 연결 확인
  • 고객 지원팀에 연락
Q: 다른 기기에서 할 일이 안 보여요

다음을 확인하세요:

  • 같은 이메일로 로그인했는지 확인
  • 인터넷에 연결되어 있는지 확인
  • 앱을 새로고침 또는 다시 시작
  • 몇 분 기다렸다가 다시 확인 (동기화 시간)
Q: 알림이 안 들려요

다음을 확인하세요:

  • 휴대폰 설정 → 알림 → DoDo에서 알림 허용 확인
  • 소리 설정 확인 (무음 모드 아닌지)
  • 배터리 절약 모드 확인
  • 앱 권한 설정 확인
Q: 파일을 첨부할 수 없어요

다음을 확인하세요:

  • 저장 공간 충분한지 확인
  • 파일 크기 확인 (너무 크면 안 됨)
  • 앱 권한 설정 확인
  • 파일 형식 확인 (일반적인 형식 사용)
Q: 데이터를 백업할 수 있나요?

네, 가능합니다!

  • 클라우드 동기화가 자동 백업입니다
  • 같은 계정으로 로그인하면 모든 데이터 복구됨
  • 수동 내보내기 기능도 있습니다 (설정 → 내보내기)
Q: 위젯이 업데이트 안 돼요

다음을 시도하세요:

  • 위젯을 삭제 후 다시 추가
  • 앱을 열어서 데이터 새로고침
  • 설정 → 위젯 설정에서 테마 다시 적용
  • 휴대폰 재부팅
Q: 위젯 글씨가 안 보여요

테마를 변경해보세요:

  • 밝은 배경화면 → 다크 테마 사용
  • 어두운 배경화면 → 라이트 또는 투명 테마 사용
  • 설정 → 위젯 설정에서 테마 변경 가능
Q: 오프라인에서 변경한 내용이 사라졌어요

다음을 확인하세요:

  • 인터넷 연결 후 동기화 완료 확인 (초록색 체크)
  • 앱을 강제 종료하지 않았는지 확인
  • 동기화 실패 시 재시도 버튼 클릭
  • 같은 계정으로 로그인했는지 확인
