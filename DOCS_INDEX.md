## 문서 관리 가이드

### 새 문서 작성 시
1. **기능 문서**: 루트 디렉토리에 `FEATURE_NAME.md` 형식
2. **기술 분석**: `claudedocs/` 폴더에 저장
3. **임시/디버그**: 작업 완료 후 `claudedocs/archive/`로 이동

### 문서 업데이트 시
1. 버전 정보가 포함된 경우 최신 버전으로 갱신
2. 완료된 작업은 체크박스로 표시 `[x]`
3. RELEASE_NOTES.md와 FUTURE_TASKS.md 동기화 유지

### 정기 정리 (월 1회 권장)
1. 3개월 이상 미사용 문서 → archive 이동
2. 중복 문서 통합
3. 이 인덱스 업데이트
