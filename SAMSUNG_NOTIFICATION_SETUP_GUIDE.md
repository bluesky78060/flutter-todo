# Samsung Galaxy 기기 알림 설정 가이드

## 개요
Samsung Galaxy 기기에서 DoDo 앱의 알림이 정상적으로 작동하도록 설정하는 방법입니다.

## ✅ 필수 설정 체크리스트

### 1️⃣ 알림 권한 허용
- **경로**: 설정 → 앱 → DODO → 알림
- **설정**:
  - "알림 허용" 토글 켜기
  - 알림 중요도를 "높음" 또는 "긴급"으로 설정
  - "Todo Reminders" 채널 활성화 확인

### 2️⃣ 배터리 최적화 해제 (가장 중요!)
- **경로**: 설정 → 배터리 및 디바이스 케어 → 배터리 → 백그라운드 사용 제한
- **설정**:
  - DODO 앱을 "제한 안 함"으로 설정
  - "절전 앱" 목록에서 제외
  - "딥 슬립 앱" 목록에서 제외

### 3️⃣ 정확한 알람 권한 (Android 12+)
- **경로**: 설정 → 앱 → DODO → 권한
- **설정**:
  - "알람 및 리마인더" 권한 허용
  - 앱이 자동으로 요청하지 않으면 수동으로 설정

### 4️⃣ 자동 최적화 비활성화
- **경로**: 설정 → 배터리 및 디바이스 케어 → 자동 최적화
- **설정**:
  - 자동 최적화 끄기 또는
  - DODO를 예외 앱으로 추가

### 5️⃣ 잠금화면 알림 표시
- **경로**: 설정 → 잠금화면 및 AOD → 알림
- **설정**:
  - "알림 내용 표시" 활성화
  - DODO 앱 알림 표시 허용

## 🔧 문제 해결

### 알림이 전혀 오지 않는 경우
1. 위의 모든 설정을 다시 확인
2. 앱 강제 종료 후 재시작
3. 기기 재부팅
4. 앱 삭제 후 재설치 (최후의 수단)

### 알림이 늦게 오는 경우
1. 배터리 최적화 설정 재확인
2. 절전 모드 해제
3. 데이터 세이버 모드 비활성화

### 특정 시간에만 알림이 오지 않는 경우
1. 방해 금지 모드 설정 확인
2. 취침 모드 설정 확인
3. 업무 시간 설정 확인

## 📱 기기별 특이사항

### Galaxy S23/S24 시리즈
- One UI 6.0 이상에서는 "적응형 배터리"가 기본 활성화
- 며칠간 앱을 사용하여 패턴을 학습시켜야 정상 작동

### Galaxy A 시리즈
- 메모리 관리가 더 엄격함
- RAM Plus 기능이 알림에 영향을 줄 수 있음

### Galaxy Z Fold/Flip
- 멀티윈도우 사용 시 백그라운드 제한이 더 엄격할 수 있음
- 커버 스크린과 메인 스크린 알림 설정 개별 확인 필요
