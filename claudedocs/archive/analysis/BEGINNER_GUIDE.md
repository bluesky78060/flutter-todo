# 🚀 DoDo 앱 - 초보자를 위한 완벽 가이드

> **DoDo는 당신의 일상을 더 효율적으로 관리해주는 할 일 관리 앱입니다!**
>
> 이 가이드는 DoDo를 처음 사용하는 분들을 위해 만들어졌습니다.

---

## 📱 DoDo 앱이란?

DoDo는 다음을 도와주는 **스마트한 할 일 관리 앱**입니다:

✅ **할 일 관리** - 할 일을 쉽게 추가하고 완료 체크
📅 **일정 관리** - 마감일 설정 및 자동 알림
🔔 **알림 기능** - 잊지 말아야 할 일을 시간에 맞춰 알려줌
📸 **파일 첨부** - 사진이나 문서를 할 일과 함께 저장
🌐 **클라우드 동기화** - 여러 기기에서 같은 할 일 목록 사용
📊 **통계 기능** - 내 생산성을 한눈에 볼 수 있음

---

## 🎯 주요 특징

| 기능 | 설명 | 활용 예 |
|------|------|--------|
| 📝 **할 일 CRUD** | 할 일 추가, 수정, 삭제 | "운동하기", "보고서 작성" 등 |
| 🏷️ **카테고리** | 일, 생활, 취미 등으로 분류 | 일과 생활을 구분해서 관리 |
| 🔁 **반복 설정** | 매일, 매주, 매월 반복 | "매일 운동", "매주 청소" |
| 📌 **서브태스크** | 큰 일을 작은 단계로 분해 | 프로젝트를 세부 작업으로 분할 |
| ⏰ **알림 설정** | 예정 시간에 자동 알림 | 중요한 일을 빠뜨리지 않음 |
| 🤐 **스누즈** | 알림을 나중에 받기 | "5분 후 다시 알림" |
| 📍 **위치 알림** | 특정 장소에서 알림 | "카페에 도착하면 알림" |
| 📁 **파일 첨부** | 이미지, PDF, 문서 저장 | "회의록.pdf" 함께 저장 |
| 🔄 **오프라인 모드** | 인터넷 없이도 사용 | 비행기 타면서도 작업 가능 |
| 📊 **통계** | 생산성 분석 | "이번 주 완료율: 85%" |

## 🚀 시작하기

### 1️⃣ 앱 다운로드 및 설치

#### Android 사용자
1. **Google Play Store** 검색창에서 "DoDo" 검색
2. **DoDo 할 일 관리** 앱 찾기
3. **설치** 버튼 클릭
4. 앱이 설치될 때까지 기다리기

#### iOS 사용자
1. **App Store** 검색창에서 "DoDo" 검색
2. **DoDo 할 일 관리** 앱 찾기
3. **받기** 버튼 클릭
4. Face ID/Touch ID로 인증하기

#### 웹 사용자
브라우저에서 DoDo 웹사이트 방문하기

---

### 2️⃣ 계정 만들기

앱을 열면 **로그인 화면**이 보입니다.

#### 선택지 1: 이메일로 가입 (쉬움) ⭐
```
1. "이메일로 가입하기" 클릭
2. 이메일 주소 입력 (예: myemail@gmail.com)
3. 비밀번호 입력 (8글자 이상, 숫자 포함 권장)
4. 비밀번호 다시 입력
5. "가입" 버튼 클릭
```

**꿀팁**: 자주 사용하는 이메일을 쓰세요. 나중에 다른 기기에서도 같은 이메일로 로그인할 수 있습니다!

#### 선택지 2: Google로 가입 (가장 쉬움) ⭐⭐
```
1. "Google로 로그인" 버튼 클릭
2. 자신의 Google 계정 선택
3. "허용" 버튼 클릭
4. 자동으로 DoDo 계정 생성 완료!
```

#### 선택지 3: Kakao로 가입
```
1. "Kakao로 로그인" 버튼 클릭
2. Kakao 계정 선택
3. "동의 및 계속" 클릭
4. 자동으로 DoDo 계정 생성!
```

---

### 3️⃣ 첫 할 일 추가하기

로그인하면 **할 일 목록 화면**이 보입니다.

```
┌─────────────────────────────┐
│  DoDo 📝                    │
├─────────────────────────────┤
│  [+ 새 할 일 추가] ← 클릭!  │
│                             │
│  할 일 목록 (아직 비어있음)  │
│                             │
└─────────────────────────────┘
```

#### 단계별 추가하기

**Step 1: "새 할 일 추가" 클릭**
- 화면 하단의 **파란 "+" 버튼** 또는 상단의 **"새 할 일 추가"** 클릭

**Step 2: 할 일 정보 입력**
```
제목 (필수) ← "운동하기" 입력
설명 (선택) ← "30분 조깅" 입력
```

**Step 3: 추가 설정 (선택사항)**
- 🏷️ **카테고리** - "건강" 선택
- 📅 **마감일** - "2025-12-10" 선택
- ⏰ **알림 시간** - "오후 6:00" 선택
- 🔁 **반복** - "매일" 또는 "매주 목요일" 등

**Step 4: "저장" 또는 "추가" 버튼 클릭**

✅ 할 일이 목록에 추가됩니다!

### 🏷️ 카테고리 관리하기

카테고리는 할 일을 **분류하는 방법**입니다.

#### 카테고리 만들기
```
1. 설정 또는 메뉴에서 "카테고리" 클릭
2. "새 카테고리" 또는 "+" 버튼 클릭
3. 카테고리 이름 입력 (예: "일", "생활", "취미")
4. 색상 선택 (선택사항)
5. "생성" 또는 "추가" 클릭
```

#### 추천 카테고리
- 📊 **일** - 업무 관련
- 🏠 **생활** - 청소, 요리, 장보기
- 💪 **건강** - 운동, 식단
- 🎮 **취미** - 게임, 독서, 영화
- 🎓 **공부** - 학습, 숙제

### 🔔 알림 설정하기

알림은 **정해진 시간에 자동으로 알려줍니다**.

#### 알림 시간 설정
```
1. 할 일 추가/수정 화면에서 "알림 시간" 탭
2. 시간 선택 (예: "오후 3:00")
3. "확인" 또는 "저장" 클릭
4. 그 시간이 되면 핸드폰이 울립니다!
```

#### 알림 스누즈 (잠깐 미루기)
```
화면의 "스누즈" 또는 "5분 후" 버튼 클릭
→ 5분, 10분, 30분, 1시간 등 선택 가능
```

---

### 📍 위치 기반 알림

"특정 장소에 가면 알림이 울린다"는 뜻입니다.

#### 위치 알림 설정
```
1. 할 일 추가/수정 화면에서 "위치" 탭
2. 지도에서 위치 검색
   - "카페 근처" → 검색
   - "회사" → 검색
3. 원하는 위치 선택
4. 반경 설정 (기본값: 500m)
5. "저장" 클릭
```

#### 활용 예
- 회사에 도착하면 "오늘 할 일 확인" 알림
- 마트 근처에 가면 "장보기" 알림
- 카페에 도착하면 "커피 주문" 알림

---

### 🔄 반복 설정하기

**같은 일을 자동으로 반복**할 수 있습니다.

#### 반복 설정 방법
```
1. 할 일 추가 화면에서 "반복" 탭
2. 반복 주기 선택:
   - 매일 (아침마다 운동)
   - 매주 월요일 (주간 회의)
   - 매월 10일 (월급일)
   - 커스텀 (자신만의 패턴)
3. "저장" 클릭
```

#### 반복 예시
| 할 일 | 반복 주기 |
|------|---------|
| 약 먹기 | 매일 아침 8:00 |
| 피부 관리 | 매일 밤 10:00 |
| 주간 미팅 | 매주 월요일 10:00 |
| 월급 확인 | 매월 10일 |
| 생일 축하 | 매년 생일 |

#### 파일 첨부 방법

   - 📷 카메라 - 사진 찍기

- **찍은 사진** - "사진 편집" 할 일에 첨부

## 🆘 자주 묻는 질문 (FAQ)

### Q: 로그인할 수 없어요
**A**: 다음을 확인하세요:
- 이메일 주소/비밀번호가 맞는지 확인
- 인터넷 연결 확인
- 앱을 완전히 종료했다가 다시 실행
- "비밀번호 찾기" 사용

### Q: 모든 할 일이 사라졌어요
**A**: 다음을 시도하세요:
- 앱 다시 시작
- 로그아웃 후 다시 로그인
- 인터넷 연결 확인
- 고객 지원팀에 연락

### Q: 다른 기기에서 할 일이 안 보여요
**A**:
- 같은 이메일로 로그인했는지 확인
- 인터넷에 연결되어 있는지 확인
- 앱을 새로고침 또는 다시 시작
- 몇 분 기다렸다가 다시 확인 (동기화 시간)

### Q: 알림이 안 들려요
**A**:
- 휴대폰 설정 → 알림 → DoDo에서 알림 허용 확인
- 소리 설정 확인 (무음 모드 아닌지)
- 배터리 절약 모드 확인
- 앱 권한 설정 확인

### Q: 파일을 첨부할 수 없어요
**A**:
- 저장 공간 충분한지 확인
- 파일 크기 확인 (너무 크면 안 됨)
- 앱 권한 설정 확인
- 파일 형식 확인 (일반적인 형식 사용)

### Q: 데이터를 백업할 수 있나요?
**A**:
- 클라우드 동기화가 자동 백업입니다
- 같은 계정으로 로그인하면 모든 데이터 복구됨
- 수동 내보내기 기능도 있습니다 (설정 → 내보내기)

## 💡 생산성 팁

### 팁 1️⃣: "오늘의 할 일" 만들기
```
1. 아침에 앱을 열기
2. 오늘 할 일만 모아놓기
3. 우선순위 순서로 정렬
4. 하나씩 완료하기
```

### 팁 2️⃣: 카테고리로 정리하기
```
- 📊 일: 업무 관련 할 일
- 🏠 생활: 집안일
- 💪 건강: 운동, 식단
→ 각 영역을 구분해서 관리
```

### 팁 3️⃣: 반복 할 일 활용
```
- 매일 해야 하는 일은 "매일 반복"으로 설정
- 예: 약 먹기, 씻기, 운동
- 자동으로 매일 나타남
```

### 팁 4️⃣: 알림으로 잊지 않기
```
- 중요한 일에 반드시 알림 설정
- 미리 알림 (1시간 전, 30분 전)
- 위치 알림 활용 (장소 기반)
```

### 팁 5️⃣: 주간 검토하기
```
- 매주 일요일에 통계 확인
- 이번 주 완료율 체크
- 미완료된 일 정리
- 다음 주 할 일 계획
```

### 📱 홈 화면 위젯

앱을 열지 않고도 홈 화면에서 할 일을 확인할 수 있습니다.

#### 위젯 활성화하기

**Android**:
1. 홈 화면 빈 곳에 길게 터치
2. "위젯" 탭 선택
3. "DoDo" 위젯 찾아서 터치
4. 홈 화면에 추가

**iOS**:
1. 홈 화면에서 "+" 버튼 (좌상단)
2. "DoDo" 찾아서 탭
3. 위젯 선택 (캘린더 또는 오늘의 할 일)
4. "위젯 추가" 버튼

#### 위젯 모드 선택

앱 설정에서 위젯 표시 방식을 선택할 수 있습니다:

**📋 오늘의 할 일**
```
- 오늘 완료해야 할 일만 표시
- 우선순위대로 정렬
- 체크박스로 위젯에서 바로 완료 가능
```

**📅 달력 보기**
```
- 현재 달 달력 표시
- 할 일이 있는 날짜 표시
- 완료된 일과 미완료 일 구분
```

#### 위젯 설정하기

```
1. 앱 열기
2. 설정 → 위젯 설정
3. 위젯 활성화 토글
4. 표시 방식 선택 (오늘의 할 일 / 달력)
5. 저장
```

#### 위젯 팁

```
💡 홈 화면에 두 개의 위젯 동시 추가 가능
   (오늘의 할 일 + 달력)

💡 위젯을 길게 터치하면 크기 조정 가능

💡 위젯을 터치하면 앱이 자동으로 열림

💡 위젯은 자동으로 주기적으로 업데이트됨
   (오프라인 모드에서도 작동)
```
