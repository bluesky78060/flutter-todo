KEY FEATURES:

✓ Cloud Sync with Supabase
- Seamlessly sync your todos across all devices
- Real-time updates when you make changes
- Secure authentication with Google and Kakao

✓ Smart Notifications
- Set reminders for important tasks
- Samsung device optimization for reliable notifications
- Foldable device support (Galaxy Fold/Flip)
- Background notifications with WorkManager

✓ Recurring Tasks
- Set up daily, weekly, monthly, or yearly repeating tasks
- Customize recurrence patterns with flexible options
- Automatic instance generation for recurring todos

✓ Category Management
- Organize tasks with custom categories
- Color-coded categories for easy identification
- Filter tasks by category

✓ Offline Support
- Works seamlessly without internet connection
- Local database with Drift (SQLite)
- Automatic sync when connection is restored

✓ Modern UI/UX
- Clean, intuitive dark-themed interface
- Fluent Design System icons
- Smooth animations and transitions
- Responsive design for all screen sizes

✓ Multi-language Support
- Available in English and Korean
- Easy language switching in settings

✓ Data Management
- Export your data as JSON backup
- Import backups with merge or overwrite options
- Keep your data safe and portable

SAMSUNG DEVICE OPTIMIZATION:
- Automatic device detection and optimization
- Battery optimization guidance
- One UI 7/8 notification setup guides
- Cover screen notification instructions for foldable devices

Whether you're managing personal tasks, work projects, or daily routines, DoDo provides the tools you need to stay on top of everything. Download DoDo today and experience the simplicity of effective task management!
