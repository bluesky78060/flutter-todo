# 개인정보처리방침

**시행일: 2025년 1월 11일**

DoDo(이하 "앱")는 사용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수합니다.

## 1. 수집하는 개인정보 항목

앱은 다음과 같은 개인정보를 수집합니다:

### 필수 정보
- **이메일 주소**: 회원가입 및 로그인 시
- **소셜 로그인 정보**: Google, Kakao 로그인 시 제공되는 기본 프로필 정보

### 선택 정보
- **프로필 사진**: 사용자가 직접 설정 시
- **위치 정보**: 위치 기반 알림 기능 사용 시 (사용자 동의 필요)
- **캘린더 정보**: Google Calendar 연동 시 (사용자 동의 필요)

## 2. 개인정보의 수집 및 이용 목적

수집된 개인정보는 다음 목적으로 이용됩니다:

- 회원 식별 및 서비스 제공
- 할 일 데이터 저장 및 동기화
- 알림 서비스 제공
- 위치 기반 알림 (선택)
- Google Calendar 연동 (선택)
- 서비스 개선 및 통계 분석

## 3. 개인정보의 보유 및 이용 기간

- **회원 탈퇴 시**: 즉시 삭제
- **서비스 미이용 시**: 1년 경과 후 자동 삭제
- **법령에 따른 보관**: 관련 법령에서 정한 기간 동안 보관

## 4. 개인정보의 제3자 제공

앱은 사용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다.

단, 다음의 경우는 예외로 합니다:
- 사용자가 사전에 동의한 경우
- 법령의 규정에 따른 경우

## 5. 개인정보의 처리 위탁

앱은 서비스 제공을 위해 다음과 같이 개인정보 처리를 위탁합니다:

| 수탁업체 | 위탁 업무 | 보유 기간 |
|---------|----------|----------|
| Supabase | 데이터 저장 및 인증 | 회원 탈퇴 시까지 |
| Google | 소셜 로그인, Calendar API | 연동 해제 시까지 |
| Kakao | 소셜 로그인 | 연동 해제 시까지 |

## 6. 개인정보의 파기 절차 및 방법

### 파기 절차
사용자의 개인정보는 목적 달성 후 별도의 DB로 옮겨져 내부 방침 및 관련 법령에 따라 일정 기간 저장된 후 파기됩니다.

### 파기 방법
- 전자적 파일: 복구 불가능한 방법으로 영구 삭제
- 종이 문서: 분쇄기로 분쇄 또는 소각

## 7. 사용자 권리 및 행사 방법

사용자는 다음과 같은 권리를 행사할 수 있습니다:

- **개인정보 열람 요구**
- **개인정보 정정 요구**
- **개인정보 삭제 요구**
- **개인정보 처리 정지 요구**

권리 행사는 앱 내 설정 메뉴 또는 이메일(bluesky78060@gmail.com)을 통해 가능합니다.

## 9. Google API 서비스 사용자 데이터 정책

앱은 Google Calendar API를 사용하며, [Google API 서비스 사용자 데이터 정책](https://developers.google.com/terms/api-services-user-data-policy)을 준수합니다.

### Google Calendar 데이터 사용
- **수집 데이터**: 캘린더 이벤트 제목, 날짜, 시간
- **사용 목적**: 할 일과 캘린더 이벤트 동기화
- **데이터 공유**: 제3자와 공유하지 않음
- **데이터 삭제**: 연동 해제 시 즉시 삭제
